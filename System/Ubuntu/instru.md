# 文件操作  
1.新建文件\
  ``touch/gedit/vim 文件名.后缀``\
2.deb格式安装\
  ``sudo apt install ./文件名``\
3.pl格式安装\
  ``sudo ./文件名``\
4.打开root权限文件夹\
  ``sudo nautilus``\
5.host添加raw.git\
  ``185.199.108.133 raw.githubsercontent.com``\
6.添加源\
  ``sudo gedit /etc/apt/sources.list``\
