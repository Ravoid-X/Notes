### 定义
相同类型的对象序列，它们占据一块连续的内存区。
### 初始化
1. 需要指定元素的类型和元素的数量\
``int age[10];``
2. 只有在定义数组时才可以使用初始化\
``int age[4];	// 正确``\
``age[4] = {1,2,3,4}   // 错误``
3. 不能将一个数组赋值给另一个数组\
``int cards[4] = {1,2,3,4};	// 正确``\
``age = cards;	// 错误``
4. 其余初始化\
``double score[3]{1.0, 2.0, 90.0};``\
``short age[] = {1,2,3,4,5};``\
``int cards[100] = {1};	//其余为0``
5. 初始化为0\
``double score[3] = {};``\
``double score[3]{}；``
